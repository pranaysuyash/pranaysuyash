## 👋 About Me

I'm a **full-stack builder** who ships products end-to-end — from Swift menu-bar apps to Python AI pipelines. I love integrating AI into real tools that actually work.

- 🧠 **AI & Multi-Agent** systems, computer vision, speech, and semantic search  
- 🍎 **Native macOS & Flutter** apps that feel like they belong  
- ⚡ **Full-Stack** — React + TypeScript frontends, FastAPI/Node.js backends, PostgreSQL  
- 🔬 Currently exploring: real-time audio intelligence, LLM agents, on-device ML with MLX
